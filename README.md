# AI Data Annotation Portfolio

Hello! I am **Beatrice Mwenje**, a Software Engineering graduate and multilingual AI data annotator.  
This portfolio demonstrates practical experience in **AI data annotation, translation, and dataset preparation**.

## Projects Included

### 1. Sentiment Analysis
- Folder: `sentiment-analysis/`
- A Python script (`sentiment_demo.py`) creates a small dataset of sentences labeled as **Positive, Neutral, or Negative**.  
- Demonstrates **text classification, dataset structuring, and annotation workflow**.

### 2. Multilingual Translation
- Folder: `multilingual-translation/`
- CSV dataset containing translations between **English, Swahili, and Kikuyu**.  
- Demonstrates **translation annotation for low-resource languages**, valuable for NLP tasks.

### 3. Image Annotation
- Folder: `image-annotation/`
- CSV dataset labeling sample images with categories like `cat`, `dog`, `vehicle`, `tree`, `device`.  
- Demonstrates **basic image labeling and dataset creation**.

### 4. Annotation Guidelines
- Folder: `guidelines/`
- Includes `annotation-guidelines.md` detailing labeling rules and best practices.  
- Shows ability to **follow instructions, maintain consistency, and ensure data quality**.
