# Sentiment Analysis Demo

This project demonstrates a **small AI data annotation workflow** for sentiment analysis.

## What it does
- Contains 10 example sentences.
- Each sentence is labeled as **Positive**, **Neutral**, or **Negative**.
- Labels are saved in `sentiment_demo.csv`.

## Purpose
- Practice dataset creation for AI and machine learning models.
- Shows ability to **structure and annotate data**.
- Can be extended for larger datasets.

## How to run
1. Make sure Python is installed.
2. Run the script:
```bash
python sentiment_demo.py
